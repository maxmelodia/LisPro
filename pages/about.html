Sobre

Essa aplicação (LisPro), tem como objetivo exibir uma pequena listagem de produtos utilizando a tecnologia PWA.
Foi desenvolvida na pós-graduação Arquitetura de Software Distribuído, matéria "Arquitetura para Aplicações Móveis" - PUC-MINAS.
Professor: Marcos Arrais

Recursos Utilizados

  • zKit
    uma coleção de componentes para web moderna.
  • WebFlix
    um clone do aplicativo móvel netflix implementado como um aplicativo da web progressivo.
  • Web Book
    um modelo de aplicativo da web progressivo de um web book.
